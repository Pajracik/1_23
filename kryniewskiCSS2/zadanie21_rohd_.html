Identyfikatory — ID

Pojęcie Znaczenie pojęcia uzupełnia uczeń
Co znaczy skrót ID? ID oznacza identyfikator, który jest unikalnym znacznikiem dla elementu w dokumencie HTML.
Jaki jest cel stosowania ID? Cel stosowania ID to umożliwienie unikalnej identyfikacji elementów, co pozwala na ich indywidualne formatowanie i manipulację w CSS oraz JavaScript.
Jakie jest znaczenie znaku #? Znak # jest używany do definiowania identyfikatora w CSS, wskazując, że styl ma być zastosowany do elementu z danym ID.
Jak następuje wywołanie ID (wraz z wytłumaczeniem)? Wywołanie ID następuje poprzez użycie selektora z #, np. #nazwaID, co odnosi się do elementu z atrybutem id="nazwaID".
Uwagi na temat stosowania ID (trzy)
  • 1. Identyfikator może wystąpić tylko raz w całym dokumencie.
  • 2. Elementy oznaczone ID muszą być unikatowe.
  • 3. W przypadku wielokrotnego użycia lepiej jest stosować klasy.
